## Ключевые особенности экономической модели

### Трехтокенная система
1. **WRT (Wert)** - Основной токен, ценность и голос в управлении
2. **LZN (Lizenz)** - Лицензия на майнинг с фиксированной эмиссией
3. **ANT (Anteil)** - Право на производительность, внутренняя единица учета

### Двухконтурная экономика
- **Контур 1**: Безопасность и стабильность (пассивный доход валидаторов)
- **Контур 2**: Производительность и скорость (активный доход через внутренний рынок)

### Инновационные механизмы
- **Минимальное Обязательство Активности (MOA)** - связывает пассивный и активный доходы
- **Слепые аукционы блоков** - справедливое распределение права создания блоков
- **Динамическое время блока** - адаптация к сетевой активности
- **Адаптивный халвинг** - связь монетарной политики с реальной полезностью

## Роли участников

### Гости (Неверифицированные)
- Базовый функционал хранения и торговли WRT/LZN
- Точка входа для всех новых пользователей
- Возможность верификации для получения расширенных прав

### Граждане (Верифицированные пользователи)
- Автоматическое начисление ANT (10 в день)
- Продажа ANT на внутреннем рынке
- Участие в управлении протоколом через DAO
- **Доходность**: 150+ WRT в месяц при средней цене ANT

### Валидаторы (Операторы узлов)
- Активация LZN для получения доли базовой эмиссии
- Покупка ANT для участия в аукционах блоков
- Двойной источник дохода: пассивный + активный
- **Доходность**: 40,000+ WRT в день для крупных валидаторов

## Математические основы

### Формула халвинга
```
Текущая_награда = 50 / (2^floor(Номер_блока / 210000))
```

### Распределение доходов валидаторов
```
Доход_валидатора = (Активированные_LZN / Общие_LZN) × Базовая_награда
```

### Динамическое время блока
```
Время_блока = 6 × (Базовая_активность / Текущая_активность)^0.5
```

### Равновесная цена ANT
```
Цена_ANT = (Спрос - Предложение + Факторы_активности) / Эластичность
```

## Экономическая безопасность

### Защита от атак
- **Атака Сивиллы**: ZKP верификация уникальной идентичности
- **Концентрация власти**: Лимит 33% LZN на валидатора
- **Манипуляции рынком**: Лимиты ордеров и мониторинг аномалий
- **Nothing at Stake**: Система MOA и экономические штрафы

### Стабилизационные механизмы
- Буферный фонд для сглаживания волатильности
- Автоматическая корректировка параметров
- Аварийные процедуры при критических ситуациях

## Технические характеристики

### Производительность
- **Пропускная способность**: 10,000+ TPS
- **Время блока**: 3-5 секунд (динамическое)
- **Финальность**: Мгновенная (1 блок)
- **Энергоэффективность**: 99.9% экономия vs PoW

- **IBC Protocol**: Межцепочечная коммуникация
